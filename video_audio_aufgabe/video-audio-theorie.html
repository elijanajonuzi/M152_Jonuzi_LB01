Video Info

Small Fact:
The MP4, WebM, and Ogg video formats are supported by HTML.

Mp4 Video

Properties of MP4 Videos

  • most common type of video file format
  • It uses the MPEG-4 encoding algorithm to store video and audio files and text
  • offers lower definition than some others

MOV Video

Properties of MOV Videos

  • stores high-quality video, audio, and effects
  • these files tend to be quite large
  • use MPEG-4 encoding to play in QuickTime for Windows

NOT SUPPORTED

AVI Video

Properties of AVI Videos

  • works with nearly every web browser on Windows, Mac, and Linux machines
  • I offers the highest quality but also large file sizes
  • works well for TV viewing
Audio Info

Small Fact:
The MP3, WAV, and OGG audio formats are supported by HTML.

SUPPORTED

MP3 Audio

Properties of MP3 Audio

  • MPEG audio layer 3(coding format) file format
  • compressed into a small size, saves a lot of space
  • it maintains near-flawless quality of original source
  • very popular for all mobile audio-playing devices

SUPPORTED

FLAC Audio

Properties of FLAC Videos

  • Free Lossless Audio Codec
  • sophisticated file type that is lesser-used among audio formats because it often needs special downloads to function.
  • audio file compressed into a smaller size of the original file

SUPPORTED

WAV Audio

Properties of WAV Videos

  • stores waveform data
  • waveform data has an image that shows strength of volume and sound in parts of the WAV file
  • typically used on Windows systems
